# 60-second walkthrough

This starts in Strava's existing Groups and Clubs experience. North Quad Run Club already has activity, but the weekly challenge gives members a specific reason to participate together.

On the club page, the challenge asks members to complete two runs with the same clubmate. Joining leads to a suggested partner based on compatible pace, availability, and usual route. Pairing with Maya turns a static challenge into a small accountability commitment.

After the runs, the payoff appears inside the existing Club Activity Feed. Ellen and Maya's activity now moves club progress and carries a shared story for other members to react to.

The final screen is still a normal Strava activity: route, athlete, stats, gear, device, and weather. The club layer stays secondary, showing that the run completed the weekly challenge and making it easy to support both runners or return to club progress.

The recommendation is not a new feed. It is a stronger challenge mechanic that gives existing club activity more meaning.
